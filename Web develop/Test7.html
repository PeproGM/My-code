  1. Яка основна ознака, що відрізняє банк від небанківської фінансової установи?
    • Право надавати кредити
    • Право залучати вклади від фізичних і юридичних осіб та розміщувати їх на власний ризик
    • Право здійснювати обмін валют
    • Право відкривати сейфи
  2. Який рівень дворівневої банківської системи України безпосередньо працює з фізичними та юридичними особами?
    • Нульовий рівень
    • Другий рівень (комерційні банки)
    • Перший рівень (центральний банк)
    • Рівень фінансових регуляторів
  3. Який з перелічених факторів є важливим критерієм при виборі банку?
    • Розмір Військового збору
    • Ліквідність активів банку
    • Наявність фінансового продукту
    • Різноманітність нематеріальних ресурсів
  4. Для якої основної мети банки відкривають клієнтам поточний рахунок?
    • Для отримання прибутку від інвестицій
    • Для зберігання цінностей у сейфі
    • Для здійснення розрахунково-касових операцій (переказу коштів та приймання/видачі готівки)
    • Для оформлення кредиту під заставу
  5. Що з переліченого належить до основних видів банківських послуг?
    • Запобігання відмиванню коштів
    • Надання кредитів
    • Регулювання фінансових ринків
    • Стягнення податку на доходи фізичних осіб (ПДФО)
